# Hardware
- Tela tft touch st7789
- esp 32c3

# Bibliotecas
- Adafruit_GFX.h
- Adafruit_ST7789.h
- XPT2046_Touchscreen.h
- qrcode_st7789.h

# Descrição
O projeto foi idealizado com intuito de facilitar a geração de qr codes pix com valores fixos, para comercios de baixa demanda 
e baixo custo. É uma calculadora quer ao final da conta é possivel gerar um qr code com o valor para a chave pix registrada no esp32.
